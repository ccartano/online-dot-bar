# Backend (NestJS)

This directory contains the NestJS backend application.

## Vercel Deployment Settings

To deploy this backend application to Vercel, use the following settings in your Vercel project configuration:

-   **Framework Preset:** Other (or NestJS if available)
-   **Root Directory:** `backend`
-   **Build Command:** `npm run build`
-   **Output Directory:** `dist`
-   **Install Command:** `npm install`

Make sure your environment variables (from `.env`) are also configured in the Vercel project settings.
